## How to Use Git
### Check things
```bash
git status
```

### Update Status of Remote
```bash
git fetch
```

### Add Things to commit
```bash
git add <file>
```
or add all changes
```bash
git add .
```

### Create Commit
```bash
git commit -m "TEXT"
```

### Pull From Remote
```bash
git pull origin master
```

### Push commits to Remote
```bash
git push origin master
```
